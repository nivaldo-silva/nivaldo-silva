## 👨‍💻 Quem sou eu

Desenvolvedor **Java Backend** com foco em construir sistemas **robustos, escaláveis e bem estruturados**.

Trabalho com o ecossistema **Spring** para criar APIs RESTful e arquiteturas de microsserviços prontas para produção, aplicando boas práticas como **Clean Code**, **SOLID** e **Domain-Driven Design**.

**O que eu entrego:**
- ✅ APIs RESTful documentadas e testadas
- ✅ Microsserviços com comunicação via Kafka/REST
- ✅ Integração com bancos relacionais e NoSQL
- ✅ Deploy containerizado com Docker e AWS
- ✅ Código limpo, revisado e de fácil manutenção

📍 Recife, PE — Brasil &nbsp;|&nbsp; 🎓 ADS — Estácio (2024)
